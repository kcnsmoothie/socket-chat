# winsock2를 활용한 C++ 소켓 통신 구현

## 기능 구현 사항

### 서버 (Server)

1. **Winsock 초기화**

   * Winsock 라이브러리를 초기화합니다.
2. **리스닝 소켓 생성**

   * 클라이언트의 연결 요청을 수신하기 위한 소켓을 생성합니다.
3. **IP 주소 및 PORT 번호 바인딩**

   * 지정한 IP와 PORT 번호를 소켓에 할당합니다.
4. **클라이언트 요청 리스닝**

   * 클라이언트의 연결 요청 대기 상태로 전환합니다.
5. **클라이언트 연결 수락**

   * 클라이언트의 연결 요청을 수락하고 통신용 소켓을 생성합니다.
6. **데이터 송수신**

   * 클라이언트와 데이터를 주고받습니다.
7. **소켓 종료**

   * 연결 종료 및 자원 해제 작업을 수행합니다.

### 클라이언트 (Client)

1. **Winsock 초기화**

   * Winsock 환경을 초기화합니다.
2. **소켓 생성**

   * 서버와의 통신을 위한 소켓을 생성합니다.
3. **서버 연결 (IP, PORT)**

   * 서버의 IP 주소와 PORT 번호로 연결을 시도합니다.
4. **데이터 송수신**

   * 서버와 데이터를 주고받습니다.
5. **소켓 종료**

   * 연결 종료 및 자원 해제 작업을 수행합니다.
